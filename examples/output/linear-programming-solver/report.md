## 四、下一步策略建议（可执行）
1. **情景压力测试**：将 `c_duration_exposure`、`c_high_vol_assets_cap` 分别收紧/放宽 5%-15%，评估收益弹性与风险代价。
2. **收益敏感性分析**：对关键资产与负债的日利率路径做 ±20~50bp 扰动，识别净收益最敏感标的。
3. **约束边界治理**：对长期不绑定约束（如单一资产A15上限、批发融资上限）评估是否需要动态阈值，而非固定阈值。
4. **运营执行建议**：维持当前低扰动调仓节奏；若市场利差扩大，可优先在不突破久期与高波动上限前提下增加高性价比资产敞口。
5. **报告机制升级**：将本报告纳入周度管理例会，固定输出“绑定约束榜单 + 收益分解 + 组合迁移图”。
